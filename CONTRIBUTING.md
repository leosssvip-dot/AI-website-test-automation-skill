# Contributing

Thanks for improving AI Website Test Automation Skill.

Good first contributions:

- Add real-project examples that preserve source evidence and redact sensitive data.
- Improve runner templates for Playwright, Cypress, Selenium, WebdriverIO, Vitest, or Testing Library.
- Strengthen coverage, flaky triage, browser-smoke, visual, accessibility, performance, or provider/live testing guidance.
- Fix helper script bugs with focused tests.

Please keep changes scoped, avoid committing secrets or customer data, and run:

```bash
npm run validate
```

Open an issue first for large scope changes, new automation frameworks, or behavior that changes the public skill contract.
